# Contributing

Thanks for helping improve the BDFD HTTP Guide.

## How to contribute

- **Fix a typo or mistake**: edit the file and open a PR.
- **Improve an example**: make it clearer or add a new one.
- **Add a missing function**: follow the existing format (syntax, what it does, examples, related functions).

## Format

Every function page follows this structure:

- What the function does (1–2 sentences)
- Syntax (code block)
- What happens (step by step)
- Examples (1–2, each with explanation)
- Common uses
- Related functions to check next

Match the tone and style of the existing pages.

## PR checklist

- [ ] Content is accurate (check against the [official BDFD wiki](https://bdfd.wiki))
- [ ] Examples actually work
- [ ] No broken internal links

## Questions

Open an issue if you are unsure about something before writing a PR.
